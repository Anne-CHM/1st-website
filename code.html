伊隆·马斯克

Elon Musk

这个世界最疯狂的人

伊隆·马斯克至今为止的创业经历:

  • Zip2

    • 马斯克和弟弟Kimbal Musk一起创建了Zip2，一个为新闻出版业的提供的网络“指南”。1999年，Compaq的AltaVista部门以3.07亿美元现金和3400万美元股票期权收购了Zip2，马斯克从中得到了2200万美元。

  • X.com和贝宝

    • 1999年3月，马斯克与他人合伙创办了X.com，一个提供线上金融服务和e-mail支付的公司。一年后，X.com和Confinity合并，并最终在2001年更名为PayPal。2002年10月，马斯克以15亿美元的价格将PayPal转卖给了全球最大的网上商店——电子港湾（eBay）。

  • 太空探索科技公司

    • 2002年6月，马斯克成立了第三家公司并兼任CEO和CTO：SpaceX（Space Exploration Technologies）。这是一家私人太空发射公司，长期的目标是实现真正的太空旅行。SpaceX获得NASA16亿美元的合同，为NASA承担了12次太空发射任务。2012年5月25日，SpaceX发射了一枚两级火箭，将一艘名为龙飞船的太空飞船送向国际空间站，开启了太空私营化的时代。目前，SpaceX已经成功发射可以重复使用的猎鹰九号。火箭变得可以重复使用，大幅降低成本。

    Musk和Bolden在感谢超过150名的SpaceX员工（SpaceX的工厂总共有3000名员工。）
  • 特斯拉汽车

    • 特斯拉汽车公司以电气工程师和物理学家特斯拉命名。马斯克是特斯拉汽车的合伙创始人和总设计师，并带领设计了第一款电动跑车Tesla Roadster。这款车已在31多个国家销售超过2300辆。特斯拉在2012年6月开始其四门轿车Model S的生产，并于2012年2月9日，推出其第三款电动汽车Model X。ModelX已于2015年9月份开始交车。最新一款Model 3在2016年3月31日发表，预计在2017年底交车，售价约35000美元起。
    特斯拉汽车在游戏极品飞车，狂野飙车，GT赛车和极限竞速系列中都有出现。
    特斯拉还为戴姆勒Smart EV和奔驰A系提供电动动力系统，为丰田RAV4提供电动系统。马斯克还成功让两家公司成为特斯拉汽车公司的长期投资者。
    特斯拉主要负责总体经营战略，旨在向大众市场的消费者提供负担得起的电动汽车。

    马斯克在Model X的发布会上
  • SolarCity

    • SolarCity成立于2006年，创办人为Lyndon Rive及Peter Rive。成立太阳能公司这个概念主要是源自Lyndon的表兄伊隆马斯克，而Elon Musk亦成为了SolarCity的主席，Lyndon担任首席执行官。
    SolarCity是全美国最大的太阳能发电公司，专门发展家用及商业光伏发电项目，总部位于美国加利福尼亚州圣马刁。SolarCity为美国太阳能发电系统供应商龙头，提供系统设计、安装以及融资、施工监督等全面的太阳能服务，以低于电力公司的价格向客户供电。
    自2006年成立以来，SolarCity发展迅速，并于2012年12月在美国纳斯达克上市，融资9200万美元，股票代号SCTY。
    除了太阳能业务外，SolarCity亦跟特斯拉汽车合作，提供家用能源储备产品Powerwall，配合太阳能板使用。2016年11月17日，特斯拉汽车以26亿美元收购SolarCity，为太阳能业史上最庞大的交易，SolarCity亦正式成为特斯拉汽车旗下的公司。



  • 此外，马斯克还提出了：
    1. 超回路列车

      2. 超回路列车是预定在美国建造的一套高速运输系统，目前正在研发中的阶段。灵感来自于伊隆·马斯克觉得北加州高速铁路的工程缓慢。这套系统的运输舱运行于减压的管路中，以太阳能供应的电磁悬浮原理轨道中行驶，运输舱则使用线性感应马达和空气压缩机推进。
      2015年10月下旬，超回路列车运输科技（Hyperloop Transportation Technologies，HTT）正式宣布将在2016年初开始第一阶段的工程，将在旧金山至洛杉矶间的中段位置，加利福尼亚州金斯郡境内建造一段耗资60亿美元、可运行的正式轨道。目前，从洛杉矶到旧金山需要1个小时的飞行，全程5个半小时的车程，坐巴士到达需要7小时，火车则是12个小时。而超回路列车建成以后，这段距离仅需35分钟。

    2. OpenAI

      3. 马斯克在2015年12月份宣布OpenAI的创立，OpenAI是一家非盈利的人工智能研究企业，旨在通过一种安全的、对人类有益的方式发展人工智能。
